UltraDesign V1.0(44)

	Key bindings which are used by other functions inside UltraDesign
	(the arrows, the escape, and return) will not work as keyboard
	abbreviations if they are placed in the Ultradesign.menus file.
	
	Other keys which have been setup as Amiga Hotkeys in other programs
	on your machine may keep them from working as abbreviations in
	in Ultradesign.menus.
